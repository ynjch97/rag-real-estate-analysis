Question: NAI형은 어떻게 만들고 어떤 의미인가요?

Answer: NAI형은 동사의 부정형을 만들 때 사용합니다. 그룹1 동사는 어미의 "u" 발음을 "a" 발음으로 바꾸고 "ない(nai)"를 붙입니다(예: 買う(kau) → 買わない(kawanai/사지 않다)). 그룹2 동사는 어미 "ru"를 빼고 "ない(nai)"를 붙입니다(예: 食べる(taberu) → 食べない(tabenai/먹지 않다)). 그룹3 동사는 "する(suru)" → "しない(shinai)", "来る(kuru)" → "来ない(konai)"로 변합니다. NAI형은 행동을 하지 않음을 표현할 때 사용합니다.
